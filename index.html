Seccion: define una seccion

Article: contenido independiente detro de seccion

Aside: contenido secundario (generalmente anuncios)

p: párrafo

hr: linea de separación

blockquote: cita larga (con margenes)

div: se usa para definir una seccion y darle la misma clase

span: contenedor de texto

em: cursiva; en css oblique

strong y b: negrita; en css font-weight: bold

small: para hacer mas pequeño un texto; en css font-size: smaller

s: texto taxado; en css text-decoration: line-through

time: para indicar una fecha 10 de enero de 2021

sub: subindice sup: superindice

i: tipo de fuente italic; en css font-style: italic

mark: fondo texto amarillo; en css background-color: yellow

cite: es una cita

q: entrecomillado

dfn: no se muy bien q es

abbr: representa una abreviacion y al pulsar sobre ella aparece ejemplo: HTML

img: imagenes

Propiedades imagenes

float: situar una imagen, por ejemplo a la izquierda: float: left;

clear: hace que no haya elementos flotantes a la derecha, izquiera o ambos ejemplo: clear: left; o clear: both;>

para centrar la imagen o bien la ponemos dentro de un texto con la propiedad text-align: center; o con display: block; margin: auto;

Eventos de teclado:

onkeydown (cuando se pulsa una tecla)

onkeypress (cuando se pulsa y despulsa una tecla)

onkeyup (cuando se despulsa una tecla)

Eventos de formulario:

onblur (cuando un elemento deja de ser seleccionado)

onchange (cuando un elemento cambia)

onfocus (cuando se selecciona el elemento)

onreset (cuando hacemos reset al formulario)

onselect (cuando un elemento se selecciona

onsubmit (cuando un formulario se envia)

Eventos de ratón:

onclick (cuando se hace clic)

ondblclick (cuando se hace doble clic)

onmousedown (cuando se pulsa un boton del ratón)

onmousemove (cuando se mueve el ratón)

onmouseout (cuando el cursor deja de situarse sobre el elemento)

onmouseover (cuando el ratón se situa sobre el elemento)

onmouseup (cuando se despulsa un boton del ratón)
